Virgo

Virgo is a comprehensive team blogging tool for Ruby on Rails

Installation Guide

1) Add Virgo to your Gemfile

gem 'virgo'

2) Run bundler

$ bundle

3) Run the Virgo install generator

$ bin/rails generate virgo:install

4) Migrate

$ bin/rake db:migrate

Migrations are pulled from the gem itself (so you won't find Virgo migration files cluttering your app's db/migrate directory).

The Virgo install generator by default will mount the Virgo engine at the root URI ("/"). We do this because we assume you are running your blog application standalone and not at a sub-uri of an existing site, though you can certainly do that (just change the mount URI to something different, like "/blog").

The Virgo install generator will also drop several (non-admin) views into "app/views/virgo" in your project tree. These are minimal templates meant to start you out and give you an idea of the default schema. We assume you will customize them deeply.
